Document Organizer Automation
Este projeto consiste numa ferramenta de automação de sistema de ficheiros desenvolvida em Python. O objetivo é organizar automaticamente diretórios desordenados, movendo ficheiros para pastas específicas com base nas suas extensões ou categorias.

Objetivo do Projeto
Eliminar a tarefa manual de organização de documentos através de um script que gere o sistema de arquivos de forma lógica, facilitando a triagem documental e o acesso à informação.

Tecnologias e Ferramentas
Linguagem: Python 3.x

Bibliotecas: os e pathlib (bibliotecas nativas do Python para manipulação de sistema operacional)

Conceitos: Iteração de diretórios, manipulação de strings e gestão de ficheiros.

Funcionalidades
Mapeamento de Diretórios: Varredura completa de uma pasta de origem definida pelo utilizador.

Classificação Automática: Identificação de tipos de ficheiros (PDF, imagens, documentos de texto, folhas de cálculo).

Gestão de Pastas: Criação automática de diretórios de destino caso estes não existam, garantindo a integridade da organização.

Movimentação de Ficheiros: Transferência segura de ficheiros para as suas respectivas categorias.

Instruções de Execução
Clone este repositório.

Certifique-se de que possui o Python instalado (não são necessárias bibliotecas externas para este projeto).

Abra o arquivo main.py e configure o caminho da pasta que deseja organizar.

Execute o script:

Bash
python main.py

Automação de tarefas repetitivas e manuais.
